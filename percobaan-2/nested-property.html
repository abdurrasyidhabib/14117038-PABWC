nested property adalah properti yang berada di dalam selektor parent. (bersarang) Misalnya seperti contoh di bawah ini.
.title {

border: {
top: 4px solid black;
bottom: 4px solid black;
}
}

class .border berada di dalam class .title, inilah yang disebut dengan nested-property.
